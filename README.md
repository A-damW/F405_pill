# F405_pill
A F405 Pill with Black/Blue Pill footprint for the [MP2 ESC](https://github.com/badgineer/CCC_ESC), a VESC-based motor control board.

The Kicad BOM contains links to [jlcpcb](https://jlcpcb.com/) for rapid PCBA.
